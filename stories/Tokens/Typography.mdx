## Usage

To use these typography styles, simply add the corresponding class name to your element:

```jsx
<h1 className="pb-h1">This is a heading 1</h1>
<p className="pb-paragraph-md">This is a medium paragraph</p>
<span className="pb-label-sm">This is a small label</span>
<input placeholder="Placeholder text" className="pb-placeholder" />
```

Each typography token is designed to maintain consistent text styling across your application.
