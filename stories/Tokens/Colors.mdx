# Colors

Our color system is based on the tailwind default color palette

- **primary (tailwind violet)**: Primary brand color of the mumble design system
- **secondary (tailwind pink)**: Secondary brand color, used for accent purposes
- **tertiary (tailwind slate)**: The tertiary color of the application, serving the purpose to style text, background, borders, etc..

The default tailwind color palette can be found [under this link](https://tailwindcss.com/docs/customizing-colors#default-color-palette).
